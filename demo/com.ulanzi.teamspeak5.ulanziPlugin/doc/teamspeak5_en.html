Setup Guide

Connecting TeamSpeak

Launch TeamSpeak and head over to the Settings -> Remote Apps. Make sure that the Port is set to 5899

Now you should see a request sent by the Plugin, accept it.

Setting up Keybindings

After the connection is established, you need to bind each Ulanzi Deck Button to a TeamSpeak Keybinding.

(Similar to what you do with a regular Key. The Plugin just sends virtual Keys)

So set up your Buttons on the Ulanzi Deck, select the corresponding function you want the Button mapped to.

For example, you place the Mute Action on your Ulanzi Deck, then you need to select Choose on the Mute Binding in TeamSpeak and press the Button on the Ulanzi Deck. You need to repeat this for every other Key as TeamSpeak currently does not offer a way to setup the Bindings automatically!

After setting all Keybindings, you are read to talk (or not if you are muted)!

Configuration of Whisperlists

Create a Whisperlist in TeamSpeak

Place your Button on the Ulanzi Deck and input the name of the Whisperlist (This name does not need to match, but could interfere with other Whisperlist Buttons if two buttons are named the same)

Click on Choose to set the Keybinding, then press the Button on the Ulanzi Deck and select the Whisperlist on the left side beside the Keybinding.
